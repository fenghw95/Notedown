Git Bash 是在 Windows 上模拟 Linux Shell 环境的工具，通过在文件夹右键选择 "Git Bash Here" 快速启动

。核心操作包括使用 `cd` 切换目录（使用 `/` 而非 `\`）、`ls` 查看文件、`pwd` 显示当前路径，以及 Git 常用命令（`git init`, `add`, `commit`, `push`）进行版本管理。

*   **下载安装**：从官网下载 Git for Windows，安装时选择默认设置即可，推荐勾选“Git Bash Here”。
*   **启动方式**：
    *   在任何文件夹内右键点击，选择 **"Git Bash Here"**（直接进入该目录）。
    *   通过开始菜单或桌面快捷方式打开。

Git Bash 使用类 Linux 命令，以下是常用文件管理命令：

*   **查看当前位置**：`pwd`
*   **切换目录**：`cd /d/folder_name` (切换到D盘文件夹)
*   **返回上一级**：`cd ..`
*   **列出文件**：`ls` (列出目录内容) 或 `ls -ah` (列出包括隐藏文件)
*   **新建目录**：`mkdir folder_name`
*   **删除文件**：`rm file_name`

*   **初始化本地仓库**：`git init`
*   **配置用户信息**（初次安装需要）：
    
    bash
    
    ```
    git config --global user.name "Your Name"
    git config --global user.email "email@example.com"
    ```
    
*   **添加文件到暂存区**：`git add <file>` 或 `git add .` (添加所有修改)
*   **提交到本地库**：`git commit -m "提交说明"`
*   **关联远程库**：`git remote add origin <URL>`
*   **推送代码**：`git push -u origin master`
*   **拉取代码**：`git pull`
*   **克隆远程仓库**：`git clone <URL>`

*   **路径转换**：Windows 的 `C:\Users` 在 Git Bash 中应写为 `/c/Users`。
*   **快捷键**：使用 `Tab` 键自动补全路径和命令。
*   **粘贴**：右键菜单选择 Paste，或使用快捷键 `Insert` (部分配置)。
*   **查看历史命令**：`history`

*   **配置别名**：为了简化命令，可以设置别名，如 `git config --global alias.co checkout`。
*   **查看配置**：`git config --list`
